# june_view
view framework by june, divide view,action,event and state for easy to use.

##  Installation
1. If the juneflow project doesn't exist, please create it by following [this guide](https://doc.juneflow.org/).
2. open terminal in the juneflow project root directory, enter the following command.
 ```bash
 june add june_view
 ```

## Usage
you can find the usage of june_view in the [document](https://doc.juneflow.org/juneview/understand).
